# [Canvas](https://canvas-opal-sigma.vercel.app/)
Hello there Hari here,
I made an online Whiteboard. It is a tool for drawing and writing. You can use it to teach take notes and think of ideas.

I really like Excalidraw. It is easy to use and simple. So I tried to make something. My Whiteboard is fast and easy to use. You do not need to log in or sign up. You can just start drawing.

---

## ✨ Features

* 🎨 Smooth freehand drawing
* 🖌️ Adjustable brush size
* 🧽 Eraser tool
* 📑 Multiple whiteboard tabs
* ↩️ Undo & Redo
* 🎯 Selection and movement tools
* 🔍 Zoom & Pan
* 📝 Text tool
* 📐 Basic shapes
* 💾 Automatic local saving
* 📤 Export as PNG, JPEG, PDF, and SVG
* 📥 Import supported files
* 🌙 Optional Dark Mode
* 📱 Responsive design for desktop, tablet, and mobile
---

## Why I Made This

Sometimes you just need a piece of paper to draw on. I made the Whiteboard so you can draw and write without having to sign up or learn how to use a tool.

You can use the Whiteboard to teach, solve problems think of ideas or just draw. It is like a piece of paper. On your computer or phone.
